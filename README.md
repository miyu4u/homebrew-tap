# Homebrew Tap

Homebrew formulae maintained by `miyu4u`.

## Install

```bash
brew tap miyu4u/tap git@github.com:miyu4u/homebrew-tap.git
brew install miyu4u/tap/recurl
brew install miyu4u/tap/toss-invest-cli
```

## Formulae

### `recurl`

Builds the [`miyu4u/recurl`](https://github.com/miyu4u/recurl) source with Homebrew's `bun` dependency and installs the compiled `recurl` executable.

### `toss-invest-cli`

Builds the [`miyu4u/toss-invest-cli`](https://github.com/miyu4u/toss-invest-cli) source with Homebrew's `bun` dependency and installs the compiled `toss-invest-cli` executable.
